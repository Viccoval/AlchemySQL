### Учебный проект по работе с реляционной базой данных и ORM SQLAlchemy.

### Проект демонстрирует построение структуры базы данных книжного магазина, загрузку данных и выполнение аналитических запросов.

##  Цель проекта:

* Построение реляционной модели базы данных
* Работа с PostgreSQL через SQLAlchemy ORM
* Импорт данных из JSON
* Выполнение JOIN-запросов
* Фильтрация данных по параметрам пользователя

## Технологический стек: 
* Python 3
* PostgreSQL
* SQLAlchemy (ORM)
* JSON
